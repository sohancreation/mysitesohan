## 📌 Overview

**SohanOS** is not just a portfolio — it is a fully functional, browser-based **desktop operating system** built from scratch using Vanilla JavaScript, Tailwind CSS, and Three.js. It showcases the work, projects, achievements, and personality of **Sohan Fardin**, a 2nd-year Electronics & Telecommunication Engineering (ETE) student at **Rajshahi University of Engineering & Technology (RUET)**, Bangladesh.

This OS-themed portfolio demonstrates advanced frontend architecture, interactive UI design, bilingual support, AI integration, and real-time 3D backgrounds — all without any frontend framework.

## ✨ Key Features

| Feature | Description |
|---|---|
| 🖥️ **OS Window Manager** | Draggable, resizable, stackable windows with a live Dock and Top Menu Bar |
| 🤖 **AI Voice Assistant** | Integrated Vapi-powered voice assistant that answers about Sohan's work |
| 🎨 **Dynamic Themes** | 4 color themes: *Cosmic, Sunset, Forest, Ocean* with Light/Dark mode |
| 🌌 **3D Backgrounds** | Animated Three.js backgrounds: *Particles, Matrix Rain, Mesh, Sonic* |
| 🌐 **Bilingual Interface** | Real-time switching between **English 🇺🇸** and **Bangla 🇧🇩** |
| 📱 **Mobile-First** | iOS-style home screen experience on mobile with app grid |
| 🕒 **Analog Clock Widget** | Stylized analog clock with 4 visual styles (*Classic, Neon, Cyan, Minimal*) |
| 📁 **macOS-style Finder** | File explorer app to browse PDFs, certificates, and project documents |
| ✉️ **Live Contact System** | Direct message form powered by Formspree with in-app chat UI |
| 🌍 **GitHub Pages CI/CD** | Auto-deployed on every push via GitHub Actions |

## 🛠️ Tech Stack

```
Frontend       →  HTML5 · CSS3 · Vanilla JavaScript (ES6+)
Styling        →  Tailwind CSS · Custom Glassmorphism · CSS Animations
3D Graphics    →  Three.js · Canvas API
AI / Voice     →  Vapi AI SDK · Gemini API (via Proxy)
Backend Proxy  →  Node.js · Express.js
Deployment     →  GitHub Pages · GitHub Actions (CI/CD)
Design System  →  Custom OS Tokens · CSS Variables · Dark/Light Themes
```

## 📂 Project Structure

```bash
mysitesohan/
│
├── index.html              # Entry point & OS shell (boot, dock, top bar, widgets)
```

## ⚙️ Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/sohancreation/mysitesohan.git
cd mysitesohan

# 2. (Optional) Start AI Proxy Server for AI features
npm install
npm start

# 3. Open in browser
# Simply open index.html in any modern browser
# Or use VS Code Live Server extension
```

> ⚡ No build step required — it's pure HTML, CSS and JavaScript.

## 🧠 Design Philosophy

SohanOS was designed to reflect the mindset of an **engineer and builder**. Rather than a standard scrolling portfolio, it presents achievements and projects as a **stateful desktop experience** — where every interaction (boot sequence, window animations, live clock, AI voice assistant) is intentional and purposeful.

The design uses **Glassmorphism** to convey transparency and clarity in engineering thinking, while **Three.js backgrounds** represent the digital space that Sohan works within every day.
